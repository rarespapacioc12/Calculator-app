# Simple JavaScript Calculator App
---
The purpose of this project is to highlight how easily JavaScript handles manipulating something displayed on the screen with respect to the user's input.

By using event listeners, basic logic and the built-in Session Storage, in just about 250 lines of code I was able to create a fully working calculator that handles basic operations, while also taking care of arithmetic errors.

To further make use of the wide variety of tools available in JavaScript, I purposely left the HTML body and CSS file empty. Thus, even the design is fully implemented in the lone script file.
